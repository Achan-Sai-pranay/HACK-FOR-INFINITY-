# EasyAble Read – Neuroinclusive Reading Chrome Extension

**EasyAble Read** is a Chrome extension designed to make web reading easier, faster, and more accessible for everyone, especially **neurodivergent users such as people with ADHD and dyslexia**.

The extension enhances webpage content with tools that **improve focus, readability, and comprehension**, helping users interact with online text in a more comfortable way.

Built during the **HackForInfinity Hackathon**, EasyAble Read aims to promote **neuroinclusive digital accessibility** on the web.

---

# ✨ Features

## 🔍 Focus Mode

Reduces distractions by highlighting the main reading area while dimming the rest of the page, helping users stay focused on the content.

## 🧠 Bionic Reading

Improves reading speed and comprehension by **bolding the first parts of words**, guiding the reader’s eyes through the text.

## 📖 Dyslexia-Friendly Font

Switches webpage text to a **dyslexia-friendly font** to improve readability and reduce visual strain.

## 🟡 Line Highlighting

Highlights lines of text to help readers **track their position while reading**, which is especially helpful for users with ADHD or dyslexia.

## 🤖 AI Text Summarizer

Quickly generates a **short summary of webpage content** so users can understand key points without reading long articles.

## 🔊 Read Voice Aloud

Converts webpage text into **speech using text-to-speech**, allowing users to listen to content instead of reading it.

# 🚀 Installation (Developer Mode)

1. Clone or download this repository.

2. Open **Google Chrome** and navigate to:

```
chrome://extensions
```

3. Enable **Developer Mode** (top right corner).

4. Click **Load Unpacked**.

5. Select the **EasyAble Read project folder**.

6. The extension will now appear in your Chrome extensions list.
